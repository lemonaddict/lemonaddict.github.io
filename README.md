# CV & Resume Repository

This repository is dedicated exclusively to the development and maintenance of a professional CV and resume. It is **not** intended to serve as a portfolio or showcase of projects.

## Technologies Used

- [Tailwind CSS](https://tailwindcss.com/) for utility-first styling
- [Alpine.js](https://alpinejs.dev/) for lightweight interactivity

## Purpose

The primary goal of this repository is to provide a clean, modern, and easily maintainable CV and resume, leveraging the latest web technologies for optimal presentation and accessibility.
